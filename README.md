### Group Members
- Hamza Hassan (100788913)

## How to Run
1. Create and enter virtual environment
```
python -m venv venv
source venv/bin/activate # or for windows source venv/Scripts/activate
```
2. Install dependencies
```
pip install -r requirements.txt
```
3. Start app
```
python app.py
```
